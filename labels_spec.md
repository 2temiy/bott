# Схема разметки

Основные метки для своего классификатора:

- `clean` — обычное сообщение без нарушения.
- `parent_insult` — оскорбления родителей, семьи, рода, явные или завуалированные.
- `general_insult` — оскорбления без упоминания семьи.
- `threat` — угрозы физической расправы, докса, преследования.
- `spam` — повторяющиеся сообщения, скам, массовые ссылки, реферальный мусор.

Рекомендации по датасету:

- минимум 3–5 тысяч примеров на класс для первого рабочего качества;
- отдельные примеры с обходами: `м@ть`, `ма ть`, `e6aл`, `6л9`, латиница + кириллица;
- балансируй `clean` и негативные классы;
- не дублируй одно и то же сообщение сотни раз;
- держи отдельный validation/test сплит.
