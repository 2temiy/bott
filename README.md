# Our Moderation AI

Это не просто промпт, а основа для **своего AI-ядра модерации**.

Что здесь есть:

- локальный текстовый классификатор на `scikit-learn`;
- гибридный движок: `regex -> local model -> OpenAI moderation`;
- отдельный класс `parent_insult`;
- обучающий скрипт;
- пример датасета;
- готовый модуль для встраивания в Telegram-бота.

## Как это работает

1. Сначала срабатывают жёсткие правила на очевидные оскорбления родителей.
2. Потом сообщение идёт в **нашу локальную модель**.
3. Потом, если нужно, сообщение добивается через OpenAI moderation.
4. Картинки пока модерируются через OpenAI, потому что свой image-model требует отдельного датасета и обучения.

## Как обучить свою текстовую модель

```bash
pip install -r requirements.txt
python train_text_model.py
```

После этого появится файл:

```bash
models/text_moderation_pipeline.joblib
```

## Как улучшить качество

- расширить `data/train_sample.jsonl` до тысяч реальных примеров;
- добавить завуалированные написания и обходы;
- делать hard negative mining из логов чатов;
- переобучать модель после новой разметки;
- отдельно собирать датасет на `parent_insult`, потому что это твоя ключевая категория.

## Реалистично про “обучение своей ИИ”

Текстовую модель ты уже можешь сделать своей прямо сейчас.
Для **своей модели на картинки** нужен:

- размеченный датасет по `gore / sexual / clean`;
- обучение vision-модели;
- валидация false positive/false negative;
- отдельный сервер инференса.

Поэтому на практике лучше так:

- **свой текстовый классификатор**;
- **OpenAI для картинок**;
- позже — своя vision-модель как второй этап.
